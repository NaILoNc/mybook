Plugin

lodash

对于Typescript 2.0以后的版本 你想引用loadsh库

如果你执行

npm install --save lodash

使用

import _ from "lodash";

会出错 Cannot find module 'lodash'.

应该还要执行

npm install --save-dev @types/lodash

即需要把loadsh添加到devDependencies中 这时候再

import _ from "lodash";

就没有问题了，或者使用

import * as _ from "lodash";

取决于你使用的Typescript版本

rimraf node_modules
